# Glow Store

Mini katalog uygulaması.

## Özellikler
- Ürün listeleme
- Ürün detay sayfası
- Sepete ekleme (simülasyon)

## Kullanılan Teknolojiler
- Flutter
- Dart

## Çalıştırma
flutter pub get
flutter run
